# Running star.sh
Download and store the GCRC_Session2-master.zip file in your $SCRATCH directory. Then make all bash scripts executable.

```bash
unzip GCRC_Session2-master.zip
cd GCRC_Session2-master/scripts/
chmod a+x *.sh
```
